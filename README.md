# E-Commerce Sales Analytics Project

## Overview

This project analyzes an online retail dataset to uncover sales trends, customer behavior, and business insights using Python, SQL, and Power BI.

## Tools & Technologies

- Python (Pandas, NumPy, Matplotlib)
- SQL (SQLite)
- Power BI
- Excel

## Project Workflow

### 1. Data Cleaning
- Removed missing values
- Handled incorrect data types
- Removed cancelled orders
- Created Revenue column

### 2. Exploratory Data Analysis
- Revenue analysis
- Customer analysis
- Product analysis
- Country-wise sales analysis

### 3. SQL Analysis
- Revenue KPIs
- Top products
- Top countries
- Customer insights
- Monthly sales trends

### 4. Power BI Dashboard
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Revenue by Country
- Monthly Revenue Trend
- Top Products
- Top Customers

## Key Insights

- Generated over 8.9M in revenue.
- Processed approximately 19K orders.
- Served more than 4K unique customers.
- Revenue is concentrated among a small group of customers.
- Certain products contribute significantly to overall sales.
